Installez l'environnement complet et le launcher en un seul clic.

TÉLÉCHARGER (.EXE)

📦 Ce que contient cet installateur automatique :

1. Chocolatey : Le gestionnaire de paquets Windows pour automatiser proprement les outils.
2. Environnement Java : Téléchargement ou configuration automatique des dépendances d'exécution de Minecraft.
3. MC Launcher (startup.exe) : L'application finale compilée prête à l'emploi sur votre bureau.
