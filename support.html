🐛 Le launcher crash ou refuse de se lancer ?

Pour résoudre efficacement votre bug, vous devez nous fournir les lignes de logs générées par votre application.

  1. Ouvrez l'explorateur de fichiers Windows et appuyez sur les touches Windows + R.
  2. Tapez %appdata%\.minecraft_custom et validez.
  3. Cherchez la zone d'affichage ou le fichier texte généré par la console verte/noire du launcher au moment du plantage.
  4. Copiez l'intégralité du texte d'erreur et transmettez-le sur notre espace communautaire (Discord ou fil GitHub Issues).
