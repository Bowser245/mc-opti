Comment l'utiliser désormais ?

  1. Lancez ce premier script une fois pour tout configurer et créer le raccourci :

    Bash 
    chmod +x setupper.sh
source ./setupper.sh

  2. Les fois suivantes, pour relancer votre script Python directement dans son environnement virtuel sans tout retélécharger, il vous suffira d'exécuter le nouveau fichier généré :

    Bash 
    ./run.sh
